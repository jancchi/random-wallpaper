# Random-wallpaper selector

A nice tool for choosing a random wallpaper from a folder and modifying theme based on it.

## Dependencies:

hyprland, bash, swww, waybar, kitty, wallust

## Use: 
 
random-wallpaper next

random-wallpaper prev

## Logic:

Chooses a random wallpaper from a folder in formats png,jpeg,jpg,webp, displayes it and saves it into a hist.txt file. Then selects another one. When you want to go back to a previous one you can.
